# 企业资产管理系统 java1247

## 项目概述

企业资产管理系统是一款基于 Spring Boot 和 Vue 技术栈开发的管理系统。该系统致力于帮助企业高效管理资产，实现资产的合理分配与使用。

## 系统角色

### 管理员角色

- 个人中心管理
- 用户管理
- 资产分类管理
- 资产信息管理
- 资产借出管理
- 资产归还管理
- 资产维修管理

### 用户角色

- 资产借出
- 资产归还

## 技术栈

### 开发工具

- IDEA 2020.3
- VSCode

### 运行环境

- JDK 1.8
- Maven 3.6.0
- MySQL 5.7
- Node.js 14

### 服务端技术

- Spring Boot
- MyBatis-Plus

### 前端技术

- HTML
- CSS
- Layui
- jQuery
- Bootstrap
- Vue
- Axios
- Element-UI

## 核心亮点

- 前后端不分离，易于集成与维护
- 使用 Spring Boot 和 Vue 技术栈，提高开发效率
- 详尽的权限控制，满足不同角色的使用需求
- 灵活配置的资产分类，适用于多种企业场景

## 部署步骤

1. 准备运行环境：安装 JDK 1.8、Maven 3.6.0、MySQL 5.7 和 Node.js 14
2. 创建数据库并导入 SQL 文件
3. 使用 Maven 打包项目
4. 部署前端文件到 web 服务器
5. 部署后端项目到 Java 应用服务器
6. 根据实际情况配置 application.properties 文件
7. 启动应用，访问前端页面进行操作

请注意：以上内容仅供参考，具体部署步骤可能因实际情况而异。
